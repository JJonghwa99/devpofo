**[Plan] 팀을 위한 명확한 설계**
* 팀원이 코드를 읽고 맥락을 파악하는 데 드는 시간을 최소화하도록 가독성을 깊이 고민합니다.
* 감이 아닌 명확한 근거를 바탕으로 기술을 선택하며, 팀원들과의 의견 조율을 통해 최적의 결론을 도출합니다.

**[Optimize] 개발자 경험 개선**
* 사소한 비효율과 불편함을 지나치지 않고, 실질적인 개선 방안을 찾아 시스템에 적극적으로 적용합니다.
* 단순 반복 업무를 자동화하고 빌드 및 배포 과정을 간소화해, 팀이 더 중요한 비즈니스 가치에 집중할 수 있도록 기여합니다.

**[Deploy] 실제로 돌아가는 환경 구축**
* NCP와 Oracle Cloud의 무료 인스턴스를 활용해 실제 클라우드 환경에서 서버를 세팅하고 배포까지 직접 경험했습니다.
* Supabase로 PostgreSQL을 매니지드 환경에서 관리해, 인프라 관리 부담을 줄이고 핵심 기능 개발에 더 집중했습니다.
